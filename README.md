# 三检自动申报

学校每日三检（晨检、午检、晚检）一键申报工具。

## 怎么用

### 第一次用

1. 手机抓包微信小程序，导出 HAR 文件
2. 把 HAR 文件放到这个文件夹
3. 双击 `提取配置.bat`，自动提取你的账号信息
4. 双击 `三检申报.bat`，完成申报

### 以后每天用

直接双击 `三检申报.bat` 就行。

程序会自动检测今天有没有新的三检，有的话帮你申报，已申报的会跳过。

## 文件说明

| 文件 | 干嘛的 |
|------|--------|
| 三检申报.bat | 点这个申报 |
| 提取配置.bat | 第一次用点这个提取配置 |
| config.txt | 存你的账号信息 |
| 使用说明.md | 你现在看的这个 |

## 注意

- session-id 过期了要重新抓包提取
- 换微信号也要重新提取
- 只会申报三检，其他活动不管
