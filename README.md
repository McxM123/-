# 三检自动申报（众接龙）

学校每日三检（晨检、午检、晚检）一键申报工具。

## 怎么用

### 第一次用

1. 手机抓包微信小程序，导出 HAR 文件
2. 把 HAR 文件放到这个文件夹
3. 双击 `提取配置.bat`，自动提取你的账号信息
4. 双击 `三检申报.bat`，程序会引导你填写申报参数
5. 填写完成后，参数会保存到 `user_data.json`

### 以后每天用

直接双击 `三检申报.bat` 就行。

程序会自动检测今天有没有新的三检，有的话自动申报，已申报的自动跳过。**全程无需确认，一键完成。**

## 文件说明

| 文件 | 干嘛的 |
|------|--------|
| 三检申报.bat | 点这个一键申报 |
| 提取配置.bat | 第一次用点这个提取配置 |
| sanjian.py | 申报程序 |
| extract.py | 配置提取程序 |
| config.txt | 存你的 session-id |
| user_data.json | 存你的申报参数（姓名、体温等） |
| cache.json | 最近扫描位置 |
| patterns.json | 自动学习的历史规律 |

## 特性

- 10线程并发扫描，带实时滚动进度条
- **规律学习**：自动从历史记录学习日均 eventId 增长量，预测今天三检出现位置，大幅减少扫描时间（积累数据越多越快）
- 规律预测命中时直接精扫小范围；未命中时自动回退大范围扫描，不会漏
- 同一标题去重，避免服务器重复接龙导致重复申报
- 自动跳过已申报的活动，不会重复提交
- 第一次运行引导填写参数，之后自动使用
- 所有请求10秒超时，不会卡住

## 注意

- session-id 过期了要重新抓包提取
- 换微信号也要重新提取
- 只会申报三检（晨检/午检/晚检），其他活动不管
